# 📊 Business Performance Overview Dashboard (Tableau)

## 📌 Project Overview

Developed an interactive Tableau dashboard to analyze overall business performance using key metrics such as Sales, Profit, Discounts, and COGS. The dashboard enables stakeholders to monitor trends, identify profitable segments, and make data-driven decisions.

## 📈 Dashboard Features

### KPI Cards
- **Total Sales:** 118.7M
- **Total Profit:** 16.9M
- **Total Discounts:** 9.2M
- **Total COGS:** 101.8M

### Sales Analysis
- Total Sales by Product and Year
- Sales Trend Over Time

### Discount Analysis
- Total Discounts Offered per Product

### Geographic Analysis
- Profit Distribution by Country (Map Visualization)

### Segment Analysis
- Profit Contribution by Customer Segment (Pie Chart)
